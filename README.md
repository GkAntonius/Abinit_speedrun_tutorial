## Overview

This tutorial comes both in the form of notebooks
    and example scripts to run tasks and plot the results.

* **0-Setup** -- Check your environment.
* **1-Task_to_flow** -- from a single `AbinitTask` to a `Flow` (Si).
* **2-Convergence_study** --  Build your flows to find convergence parameters (GaAs).
* **3-Relaxation** -- Relaxating atomic positions and cell volume (AlN)
* **4-Phonons** -- Use existing workflows to compute phonon band structure (MgO)
* **5-Challenge** -- Put it all together.
