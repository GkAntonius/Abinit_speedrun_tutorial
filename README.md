# The Abinit Speedrun Tutorial

## About the tutorial

Let us speedrun through Density Functional Theory (DFT) calculations by making workflows and inspecting the tasks along the way.
This tutorial is intended for new Abinit users AND experienced users wanting to master automation with Abipy.

**Official Abinit & Abipy docs:**

* The [Abinit website](https://www.abinit.org)
  * Index of [input variables](https://docs.abinit.org/variables/).
  * Several [tutorials](https://docs.abinit.org/tutorial/).

* The [Abipy website](https://abinit.github.io/abipy/).
* The [abipy_book](https://abinit.github.io/abipy_book/intro.html).

## Overview

This tutorial can be followed through notebooks (`Notebooks/*.ipynb')
    and example scripts (`Examples/*.py`) that let you run tasks and plot results in the terminal shell..

* **0-Setup** -- Check your environment.
* **1-Task_to_flow** -- from a single `AbinitTask` to a `Flow` (Si).
* **2-Convergence_study** --  Build your flows to find convergence parameters (GaAs).
* **3-Relaxation** -- Relaxating atomic positions and cell volume (AlN)
* **4-Phonons** -- Use existing workflows to compute phonon band structure (MgO)
* **5-Challenge** -- Put it all together.
