# The Abinit Speedrun Tutorial

## About the tutorial

Let us speedrun through Density Functional Theory (DFT) calculations by making workflows and inspecting the tasks along the way.
This tutorial is intended for new Abinit users AND experienced users wanting to master automation with Abipy.

You can read through the tutorial in less than 30 min,
    do the exercices along the way in ~1.5h,
    and complete one of the challenges in a 3h session.

## Overview

This tutorial can be followed through notebooks (`Notebooks/*.ipynb`)
    and example scripts (`Examples/*.py`) that let you run tasks and plot results in the terminal shell.

0. **Setup** -- Check your environment.
1. **From Task to Flow** -- Run ground state and band structure calculations (Si).
2. **Building your flows** --  Build convergence study flows to find calculation parameters (GaAs).
3. **Relaxation** -- Relax atomic positions and volume to find lattice parameters (AlN).
4. **Phonons** -- Use existing workflows to compute a phonon band structure (MgO).
5. **Challenges** -- Explore new materials.
